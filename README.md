# go-test-color

![Go](https://github.com/haunt98/go-test-color/workflows/Go/badge.svg?branch=main)

Run `go test` with color.

## Install

```sh
GO111module=on go get -u github.com/haunt98/go-test-color
```

## Usage

```sh
# Simply replace go test with go-test-color
go-test-color -v ./...
```
